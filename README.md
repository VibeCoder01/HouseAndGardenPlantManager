Purpose
- Offline-first Obsidian plugin.
- Minimal interaction. Maximal utility.
- Watering and feeding guided by checks and seasons, not timers alone.
- Garden rotation awareness and frost-date planning.

How to use this pack
1. Read SPEC.md first.
2. Open the project in your editor.
3. Run `npm install` then `npm run dev` to build in watch mode.
4. Install in Obsidian by copying `manifest.json` and `main.js` to your vault's `.obsidian/plugins/houseplant-garden-manager/` folder.
5. Use the example vault files to verify logic and UI.

Notes
- British English spellings.
- No external services.
- Species presets are optional examples, not rules.
